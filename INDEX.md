## 🎨 Key Customizations

### Change Color Theme
Edit line 9 in `css/styles.css`:
```css
--primary-color: #0A74DA;  /* Change this */
```

### Update Your Information
Edit `index.html`:
- Line ~50: Your name
- Line ~410: Email
- Line ~412: Phone
- Line ~418: GitHub link
- Line ~419: LinkedIn link

### Add Profile Picture
1. Save image to `assets/images/profile.jpg`
2. Edit line ~62 in `index.html`
3. Add: `<img src="assets/images/profile.jpg" alt="Profile">`
