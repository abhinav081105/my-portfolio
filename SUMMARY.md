## 🚀 How to Use

### **1. Open Locally**
```bash
# Double-click index.html
# OR use Python server
python -m http.server 8000
# Then visit http://localhost:8000
```

### **2. Customize Content**
- Edit `index.html` for your content (name, projects, links)
- Edit `css/styles.css` for colors and styling
- Add your profile picture to `assets/images/`

### **3. Deploy Online**
- **GitHub Pages**: Free, automatic deployment
- **Netlify**: Free, drag & drop or Git integration
- **Vercel**: Free, optimized for performance

## 🎨 Customization Cheat Sheet

### Change Accent Color
```css
/* In css/styles.css, line 9 */
--primary-color: #0A74DA;  /* Change this hex code */
```

### Update Your Name
```html
<!-- In index.html, line ~50 -->
<h1 class="hero-title">Your Name Here</h1>
```

### Update Contact Info
```html
<!-- In index.html -->
Email: Line ~410
Phone: Line ~412
GitHub: Line ~418
LinkedIn: Line ~419
```

### Add Profile Picture
```html
<!-- In index.html, line ~62, replace SVG with: -->
<img src="assets/images/profile.jpg" alt="Profile">
```

### Enable 3D Animations
```html
<!-- Add before </body> in index.html -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>
```
